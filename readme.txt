atenção: é necessário ter o python instalado 

antes de iniciar o programa abra o cmd dentro desta pasta e digite o seguinte:

py -m pip install -r requirements.txt

para rodar o bot abra o cmd na pasta novamente e digite o seguinte no console:

py main.py -u [ra com digito sem o SP] -p [senha] -c [canal dos vídeos, sem ou anual]

você também pode usar argumentos como: 

-u [escolhe o uf do ra, por padrão é SP] -a [quantidade de vídeos que você quer que o bot veja, por padrão ele vê todos]

obs: não é para escrever com os []

caso você queira uma explicação do que cada argumento faz digite:

py main.py --help
